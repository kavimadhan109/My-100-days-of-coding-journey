## Statement
```
Write a program that greets the user by printing the word "Hello", a comma, the name of the user and an exclamation mark after it. See the examples below.
```
## Sample input
```
Harry
```
## Sample output
```
Hello, Harry!
```
## Code
```
# Read a string:
a = ("Hello, ")
# Print a string:
b = input()
print(a + b + "!")
```
